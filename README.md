# Notes App

A simple notes app. Uses Drift to store your notes. Assign colors for your notes. Search your notes easily.

## Features
 - Supports Android/iOS (IOS not available yet)
 - Search notes
 - List view for notes
 - Color labels for notes
 - Smooth UI
